# Sales & Revenue Analysis Dashboard

## Project Overview

This project analyzes cosmetics shop sales data using Power BI.

## Tools Used

* Power BI Desktop
* Microsoft Excel

## Dashboard Features

* Total Revenue KPI
* Total Quantity Sold KPI
* Total Products KPI
* Revenue Trend Analysis
* Product-wise Revenue Analysis
* Category-wise Revenue Analysis
* Interactive Slicers and Filters

## Files Included

* Cosmetics_Sales_Dashboard.pbix
* Cosmetics_Shop_Dataset_100Rows.xlsx
* Dashboard_Screenshot.png

## Outcome

The dashboard helps visualize sales performance, identify top-selling products, and analyze revenue trends for better business decision-making.
